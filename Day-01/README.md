# Day 01 - AI vs ML vs Deep Learning

## 📅 Date: March 2026

## 📖 What I Learned
- Difference between AI, Machine Learning, and Deep Learning
- AI is the broad concept
- ML is a subset of AI
- Deep Learning is a subset of ML

## 🔑 Key Takeaways
- AI = making machines smart
- ML = machines learning from data
- Deep Learning = ML using neural networks
